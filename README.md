# NeonGame
Unity introduction assignment.

Player controls:
  - Player 1:
    'W' -> move up
    'A' -> move left
    'S' -> move down
    'D' -> move right
    'Spacebar' -> fire
    
 - Player 2:
   'Up' -> move up
   'Left' -> move left
   'Down' -> move down
   'Right' -> move right
   'Left-click' -> fire
